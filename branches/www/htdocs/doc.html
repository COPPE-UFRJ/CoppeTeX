Thesis template

It is not difficult to encounter thesis and dissertations with some hundreds of pages. The maintenance of quite a big document may be tedious if it is contained in only one file. The best way of organizing a large manuscript is to split it into files of smaller sizes. In this case, it is usual to have a main file issuing LaTeX commands to include chapters and appendices, each of which has its own file.

To avoid our users redo the same work again and again and again, we have created a ready-to-use thesis template using the ideas cited above. It is not required to install the CoppeTeX bundle, since all necessary files were packed together. Just decompress it, fill in the template with your data and then run LaTeX, BibTeX, Makeindex and whatever you need, as described in the user's manual.
